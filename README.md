Swiggy Sales Analytics Dashboard
Power BI & Excel | Data Analysis | Business Insights

This project showcases a Swiggy Sales Analytics Dashboard built using Power BI with data collected and cleaned in Excel. The dashboard provides a complete analysis of sales trends, customer ratings, popular cities, food types, and state-wise performance.

Project Overview

This project analyzes Swiggy’s business performance using an interactive Power BI dashboard.
The data was cleaned, transformed, and structured in Excel, then visualized in Power BI.

Key Objectives:

Analyze monthly, weekly, and daily sales performance

Track customer ratings and average order value

Understand food-type preferences

Compare city and state-wise performance

Visualize quarterly sales and order trends

Provide actionable insights for decision makers

Insights & Observations

Bengaluru contributes the highest city-level sales.

Weekends drive the strongest performance.

Food Type 1 dominates customer preference (approx. 65%).

All quarters have consistent customer ratings of 4.3+.

North, South, and West India show strongest regional performance.
